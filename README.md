# Web Kuis Sederhana

Project ini adalah web kuis sederhana berbasis HTML, CSS, dan JavaScript
yang dibuat untuk latihan logika dan DOM manipulation.

Fitur
- Pertanyaan pilihan ganda
- Penilaian otomatis
- Tampilan interaktif

Teknologi
- HTML
- CSS
- JavaScript

Algoritma / Cara Kerja
1. Soal ditampilkan ke halaman
2. User memilih jawaban
3. JavaScript mengecek jawaban
4. Skor dihitung
5. Hasil ditampilkan

Cara Menjalankan
Buka file `index.html` menggunakan browser
